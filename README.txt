همه توضیحات یکجا
یک برنامه ویندوزی ساده، برنامه را در پوشه ای که میخواهید ثابت بماند (میخواهید برنامه آنجا نصب شود) کپی کنید، کافیست آن را یک بار اجرا کنید(نیاز به امتیاز مدیر ندارد) برنامه بصورت خودکار نصب میشود و هربار که ویندوز شروع به راه اندازی کند بصورت خودکار اجرا میشود.
بصورت پیش فرض هر 20 دقیقه که با کامپیوتر کار کنید به مدت 1 و نیم دقیقه صفحه به آرامی تاریک میشود و شمارنده معکوس بکار می افتد پس از سپری شدن زمان استراحت چشم، صفحه به آرامی به حالت اولیه بر میگردد و  دوباره میتوانید با کامپیوتر کار کنید. فایل تنظیمات برنامه یک ini ساده است که میتوانید آن را در یک ویرایشگر ساده مثل نوتپد ویندوز باز کنید و گزینه های آن را تغییر دهید وقتی تغییرات را ذخیره کنید برنامه بصورت خودکار تغییرات را لود میکند و از تغییرات جدید پیروی میکند (نیاز نیست برنامه را ریستارت کنید). در صورتی که بخواهید زمانی که صفحه بطور کامل تاریک شده است آن را به اجبار به حالت عادی برگردانید کلیدهای ترکیبی Ctrl+H را بزنید(صفحه تاریک پنهان میشود)


All Instructions at Once

A simple Windows program: Just copy the program to the folder where you want it to stay permanently (where you want it "installed"). You only need to run it once (no admin rights required), and it will install automatically. Every time Windows starts up, the program will launch automatically.

By default, after every 20 minutes of computer use, the screen will gradually dim for 1.5 minutes, and a countdown timer will appear. Once the eye-rest period is over, the screen will slowly return to normal, and you can resume working.

The program’s settings are stored in a simple .ini file, which you can edit with a basic text editor like Windows Notepad. After saving your changes, the program will automatically reload and apply the new settings (no restart needed).

If you want to forcefully return the screen to normal while it’s fully dimmed, press the Ctrl + H shortcut (the dark screen will be hidden).
